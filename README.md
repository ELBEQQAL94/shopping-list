### Simple Shopping app using Nodejs in back end and Reactjs in front end

## Packages

* body-parser.
* concurrently.
* cors.
* dotenv.
* express.
* lodash.
* mongoose.
* morgan.

### npm run server on PORT{8000}

## `GET ITEMS`

GET: /items

## `GET ITEM`

GET: /items/:id

## `UPDATE ITEMS`

PUT: /items/edit-item/:itemId

## `CREATE ITEMS`

POST: /items/create-item

## `DELETE ITEMS`

DELETE: /items/:itemId
